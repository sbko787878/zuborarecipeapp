## 🛠 使用技術スタック

### バックエンド
- **Java 21**
- **Spring Boot 3.2.x**
- **Spring Security**

### データベース / インフラ
- **MariaDB**
- **AWS (EC2)**
- **Docker** （移行・コンテナ化を計画中）

### フロントエンド
- **Thymeleaf** / **JavaScript**
- **HTML5 / CSS3** (レスポンシブ対応予定)

## ✨ こだわったポイント
1. **Java 21 の採用**: モダンな開発環境を意識し、最新のLTSバージョンを選択。
2. **AWSでの外部公開**: 自らインフラ構築を行い、Webサーバーの運用を経験。
